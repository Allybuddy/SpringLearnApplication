Message queue is made up of producer, broker (the message queue software), and consumer.
It provides asynchronous communication between different components of a system.
RabbitMO broker => exchange -> queues
Routing key: a key that the exchange looks at when deciding how to route the message to queues.
Binding: link exchange to queue
